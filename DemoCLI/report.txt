Work Item #29: User Story 4: Detailansicht der Urlaubsvorschläge
  State: To Do
  Description: Als angemeldeter User möchte ich eine Bildvorschau, einen Beschreibungstext der Reise, eine Liste von Nutzerbewertungen und eine durchschnittliche Gesamtwertung der Reise in Form einer Skala von 1-5 Sternen sehen, damit ich mir ein genaueres Bild über das Reiseziel machen kann.

Work Item #28: User Story 3: Reisevorschläge anhand Antworten anzeigen
  State: To Do
